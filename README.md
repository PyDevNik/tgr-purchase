# tgr-purchase
## Plugin (For WordPress) for Purchasing TGR with tegro.money API

# Requirements
- PHP 5+
- Python 3.7+

# Install & Build
```bash
    git clone https://github.com/PyDevNik/tgr-purchase.git
    pip install requirements.txt
```

# Setup
```bash
    Setup file - settings.json
    Add shop_id and api_key
```
